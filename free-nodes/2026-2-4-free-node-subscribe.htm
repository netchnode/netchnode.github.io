什么是Shadowrocket？

Shadowrocket是一款强大的代理软件，兼容iOS设备，允许用户通过多种代理协议（如ShadowSocks、VMess、Trojan等）实现更安全、更自由的上网体验。它能够绕过各种网络限制，保障用户的在线隐私，并提供高效的网络加速功能，成为了很多追求网络自由和隐私保护的用户首选工具。

与传统的VPN软件不同，Shadowrocket通过支持分流代理和定制规则等独特功能，让用户能够更灵活地管理网络流量，精准控制哪些内容需要通过代理服务器访问，哪些可以直接访问。通过这种方式，Shadowrocket为用户提供了一个更加安全、自由且高效的网络体验。

Shadowrocket的核心功能

1. 分流代理

Shadowrocket最大的亮点之一就是它的分流代理功能。通过这项功能，用户可以根据自己的需求精确地控制哪些流量需要经过代理服务器。例如，用户可以设置只将某些应用程序或网站的流量通过代理，其他流量则直接通过本地网络传输。这样一来，不仅可以提高网络效率，还能更好地保护隐私。

2. 规则定制

Shadowrocket支持用户根据自己的需求创建自定义规则。用户可以为不同的网站、应用程序或域名设置独立的访问规则，灵活地定义哪些流量走代理，哪些流量直连。这一功能特别适合那些需要精细化管理流量的用户，尤其是在复杂的网络环境中，能够极大提高上网体验。

3. 多协议支持

Shadowrocket支持多种常见的代理协议，包括ShadowSocks、VMess、Trojan等，能够适应不同的网络环境。在需要突破地理限制或进行安全通信时，用户可以根据自身需求选择最合适的协议。不同协议的支持使得Shadowrocket能够在各种情况下为用户提供高效、安全的上网方式。

4. 简便的分享功能

Shadowrocket还提供了方便的配置文件分享功能，用户可以将自己配置好的代理节点和规则与他人共享，避免了重复设置的麻烦。通过分享配置，用户可以更轻松地在不同设备之间切换，快速实现代理服务的切换。

5. 加密和安全性

Shadowrocket通过对数据进行加密，能够有效保障用户的上网安全。在公共Wi-Fi环境中使用时，用户的网络数据可以得到充分保护，防止个人信息泄露或遭受黑客攻击。此外，通过使用高质量的代理节点，用户还能够获得更高的上网速度。

FAQ

PassWall2 的延迟测速功能如何提高网络体验？
PassWall2 可以定期测速节点延迟，并结合策略组自动选择最优节点。这样在访问网站或视频时，可以始终使用延迟最低的线路，提高网络稳定性和速度。
Nekoray 支持节点批量更新吗？
支持。Nekoray 可通过订阅 URL 批量下载最新节点并替换旧节点。结合定时更新功能，可保持节点列表最新，减少手动维护和配置工作。
PassWall2 如何实现智能分流？
PassWall2 通过自定义规则文件控制流量走向。可根据目标域名、国家或 IP 范围自动判断走代理或直连。用户还能加载社区规则，实现类似 Clash 的高效路由分流。
VMess AEAD 加密能防止哪些风险？
AEAD 加密可防止数据篡改、重放攻击和流量特征泄露。在高封锁环境下使用，有助于保持节点长期可用，保证数据安全和连接稳定性。
SSR 的混淆和协议组合如何选择？
根据网络封锁情况选择适合的混淆和协议组合，例如 tls1.2_ticket_auth 模拟 HTTPS 流量。不同组合对抗 DPI 和封锁能力不同，需要测试后选择最稳定和高效的方案。
Potatso Lite 节点手动切换有什么优势？
手动切换节点可以让用户根据网络延迟和用途选择最合适的节点，适合在节点不稳定或特殊访问需求下使用。虽然不自动，但可以灵活调整，提高访问效率。
Sing-box 精确端口分流如何提升游戏连接？
Sing-box 可按端口设置流量分流规则，将游戏流量分配到低延迟节点。结合策略组使用，可保证游戏连接稳定，减少延迟和掉线，提高流畅度和网络体验。
Mellow 使用透明代理时如何避免 DNS 泄露？
在 Mellow 中启用 fake-ip 或内置 DNS 解析功能，并将上游 DNS 指定为 DoH/DoT 服务，确保所有 DNS 请求通过代理转发而非本地解析，从而避免 ISP 或本地网络捕获实际访问域名。
Surfboard 支持哪些类型的节点导入？
Surfboard 支持 VMess、VLESS、Trojan、Shadowsocks 等节点类型。用户可通过 YAML 或 JSON 文件导入，配置策略组和规则，实现自动分流和节点管理。
QuantumultX 支持哪些类型的分流规则？
QuantumultX 支持域名、IP、端口、协议和用户自定义脚本分流。结合策略组使用，可实现精准流量控制，如广告屏蔽、视频加速和游戏优化，提高访问速度和体验。
